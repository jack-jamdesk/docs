New Feature: AI-Powered Content Freshness Detection

Keeping documentation up to date is one of the biggest challenges for growing teams. Our new AI-Powered Content Freshness Detection feature automatically identifies pages that may contain outdated information and surfaces them for review.

The system analyzes documentation for signals such as old version references, deprecated APIs, broken links, stale screenshots, and content that hasn’t been updated in a long time. Pages that may need attention are assigned a freshness score and added to a review queue.

Key benefits include:

- Identify outdated documentation before customers find it.
- Reduce support tickets caused by stale information.
- Prioritize documentation maintenance efforts.
- Improve trust in your documentation portal.
- Keep AI-powered search results more accurate.

To get started, enable Content Freshness Detection from the Documentation Settings page. The system will begin scanning your documentation and generate an initial report within a few minutes.

This feature is available now for Pro and Enterprise customers.
